Sanat Galerisi Web Sitesi

Bu proje, bir müze/sanat galerisi konseptiyle geliştirilmiş basit bir web sitesidir. Site, 
müze hakkında genel bilgiler sunmak ve sergilenen eserleri kullanıcıya tanıtmak amacıyla hazırlanmıştır.

Proje Amacı
Temel web geliştirme becerilerini (HTML, CSS) uygulamak
Statik bir web sitesi yapısını öğrenmek
İçerik odaklı sayfa tasarımı oluşturmak

İçerik
Müze hakkında bilgilendirme bölümü
Sanat eserlerinin sergilendiği alan
Basit ve düzenli bir sayfa yapısı
bulunmaktadır.

Kullanılan Teknolojiler
HTML
CSS
